# Library-Management-System
GUI project of Library Management System in Python using Tkinter and SQL

## Functionalities  
           - add books
           - delete/update books
           - search books
           - issue book
           - return book
           - student activity
         
## Installations
Install the following packages in cmd before use.
```
pip install python-tk
pip install pillow
```


## Technologies used 
* Python and SQL
* Tkinter for frontend
